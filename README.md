# NoTo0870

This package provides a PHP API to obtain alternatives to UK 0870 numbers from http://www.saynoto0870.co.uk/.

### Requirements

- PHP 5.5

### Installation

**Composer**

Run the following to include this via Composer

```shell
composer require rubenarakelyan/NoTo0870
```
